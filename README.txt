Folder Guide
--------------
1. If you're looking for general information on the server plans, check the 'General' folder.

2. If you want info on the DarkRP jobs, check the 'Jobs' folder.

3. Looking for what addons/mods or maps could be used? Check the 'Mods' folder.

4. Miscellaneous info is in the 'Other' folder.

5. Info on Plugins is in the 'Plugins' folder.
